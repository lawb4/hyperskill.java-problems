Multiply user integers and check parity

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that reads two integers from the user, multiplies them together, and then prints out the result. The program should also print out whether the result is even or odd.
Sample Input:
4
6

Sample Output:
24
even


Sample Input:
3
5

Sample Output:
15
odd



Memory limit: 256 MB
Time limit: 5 seconds
