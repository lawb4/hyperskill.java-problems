Performing operations on string digits

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Your task is to create a Java solution that accepts a single string input from the user, which should be a digit. Then, depending on the value of the input, two operations will be performed: If the last character of the string is a digit and the digit is even, then the program will transform the digit from a String data type to an Integer data type, then multiply it by two. If it's an odd digit, transform it to an Integer and add 5 to it. Print the result.
Sample Input:
4

Sample Output:
8


Sample Input:
7

Sample Output:
12



Memory limit: 256 MB
Time limit: 5 seconds
