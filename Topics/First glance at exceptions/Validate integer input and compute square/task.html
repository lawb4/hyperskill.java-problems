Validate integer input and compute square

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that reads an integer from the user. If the integer is negative, throw an exception with the message "InvalidInputException: Input must be non-negative.". If the integer is positive, print the square of the integer. If the integer is zero, print "Input is zero.".
Sample Input:
5

Sample Output:
25


Sample Input:
-3

Sample Output:
InvalidInputException: Input must be non-negative.



Memory limit: 256 MB
Time limit: 5 seconds
