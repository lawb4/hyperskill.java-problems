Logical operations on boolean inputs

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that takes two boolean values as input and outputs the result of the logical AND, OR, and NOT operations on those values. Input: The program should read two boolean values (either true or false) from the user, each on a separate line. Output: Print the result of the logical AND, OR, and NOT operations on the input values, each on a separate line.
Sample Input:
true
true

Sample Output:
true
true
false


Sample Input:
false
true

Sample Output:
false
true
true



Memory limit: 256 MB
Time limit: 5 seconds
